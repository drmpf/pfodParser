pfodParser

The pfodParser library handles commands sent from pfodWeb and the Android pfodApp.

pfodWeb.html (included in this library) displays your GUI and runs in your brower. It connects to any Arduino board via Serial. It can also connect via BLE, TCP/IP Socket or HTTP. See the pfodParser/examples/pfodWeb Arduino examples in this library and the pfodWeb Installation and Tutorials
pfodApp (Android app) supports WiFi, BLE, Bluetooth and SMS connections.
The entire GUI is completely controlled by your micro's sketch.

This library includes:-

  • pfodParser, parses pfod commands and serves the GUI
  • pfodWeb, browser version of pfodApp that connects via Serial, BLE or HTTP to display interactive GUI's
  • pfodSecurity, alternative to pfodParser that adds 128bit sercuity
  • pfodDwgs, classes for sending dwg commands to create interactive GUI's

pfodWeb

The pfodWeb sub-directory contains installs for Windows, MacOS and Linux. See pfodWeb Installation
No local server is required. No additional downloads are required. You can run pfodWeb on a completely isolated network with no internet access.

The pfodWeb/data sub-directory contains pfodWeb as .gz files suitable for serving from your microprocessor for a completely self-contained deployment.
The pfodWeb/docs sub-directory contains the docs for pfodWeb.

pfodWeb is open-source. The source is available from pfodWeb_src.zip and github hosted at pfodWeb_src
